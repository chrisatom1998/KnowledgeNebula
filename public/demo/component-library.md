This document lists our reusable, core React components.

## Core Component Inventory
- **Button**: Custom wrapper supporting loading spinners and icons.
- **Modal**: Built using React portals, handles focus trapping and Esc closures.
- **Dropdown**: Fully accessible select replacements.
- **TextInput**: Combines labels, error states, and input fields.

## Adding Components
Do not create custom components that duplicate the functions of the core library.
