Before writing code for major features, engineers must draft a Technical Specification document.

## Required Sections
1. **Overview**: High-level goal.
2. **System Architecture**: Data flow diagrams.
3. **Data Model**: SQL schemas or API changes.
4. **Security Risks**: Threat evaluation.
5. **Rollout Plan**: Feature flags details.
