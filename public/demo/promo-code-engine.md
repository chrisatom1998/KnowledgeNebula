The promo code engine manages promotional discounts and coupon verification.

## Rule System
Coupons specify:
- **Scope**: Applied to billing plans or specific items.
- **Limits**: Maximum usage counts and expiration dates.
- **Compatibility**: Can coupons be stacked on top of existing discounts?
