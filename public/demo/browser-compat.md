This document defines our target browser configurations.

## Supported Browsers
We support the last 2 versions of major browsers:
- Google Chrome
- Apple Safari
- Mozilla Firefox
- Microsoft Edge

## Polyfills
We use Autoprefixer for CSS. Avoid using experimental web APIs (e.g., CSS Container Queries) unless polyfilled or checked with progressive enhancement patterns.
