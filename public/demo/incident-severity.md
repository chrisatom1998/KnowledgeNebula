This matrix defines incident priority levels and response times.

## Priority Levels
- **Severity 1 (P0)**: System offline for all users. Response: immediate page.
- **Severity 2 (P1)**: Core feature degraded (e.g., checkout failing). Response: 15-minute SLA.
- **Severity 3 (P2)**: Minor UI bug. Response: next business day.
