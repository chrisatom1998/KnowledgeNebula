To ensure appropriate handling of data resources, Nimbus Labs classifies all data into four distinct confidentiality levels.

## Classification Levels
1. **Public**: Marketing materials, open documentation.
2. **Internal**: System architecture diagrams, meeting notes.
3. **Restricted**: Codebases, configuration details, cost reports.
4. **Confidential**: Personally Identifiable Information (PII), API keys, billing secrets.

## Safe Storage Practices
Confidential data must be encrypted at rest and masked in logs.
