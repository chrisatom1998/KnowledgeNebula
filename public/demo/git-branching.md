Nimbus Labs follows a Trunk-Based Development model.

## Rules
- Direct pushes to the `master` branch are blocked.
- Create feature branches named `feature/[name]` or `bugfix/[name]`.
- Merging to `master` requires an approved Pull Request and green CI builds.
