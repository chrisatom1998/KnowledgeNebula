To ensure defense-in-depth, Nimbus Labs contracts with an external security firm to perform independent network and application penetration testing annually.

## Scope of Testing
The testing includes:
- API endpoint authentication bypass checks.
- AWS network boundaries and IAM privilege escalations.
- Social engineering and credential harvesting audits.

## Remediation Tracker
All findings are tracked in our security backlog and must be resolved before SOC2 audits can be completed.
