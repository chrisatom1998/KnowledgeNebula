Our design system ensures visual consistency and user-friendly patterns across all Nimbus Labs application portals.

## Color Tokens
We utilize HSL color variables:
- **Primary Tint**: `hsl(220, 90%, 56%)` (Classic Royal Blue)
- **Background**: `hsl(240, 20%, 4%)` (Deep Nebula Black)
- **Accent**: `hsl(280, 80%, 65%)` (Luminous Purple)

## Typography and Spacing
Font families are restricted to Inter (Sans-serif) and Outfit (Headers). Spacing utilizes a strict 4px grid system (`padding: 8px`, `12px`, `16px`, etc.).
