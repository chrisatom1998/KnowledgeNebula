We build our interfaces to be accessible to all users. All web components must adhere to the WCAG 2.1 AA specifications.

## Essential Criteria
1. **Keyboard Nav**: All interactive elements must be focusable using the `Tab` key.
2. **Aria Labels**: Screen readers must understand icon-only buttons via `aria-label`.
3. **Contrast Ratio**: Texts must have at least a 4.5:1 contrast ratio against the background.
4. **Focus States**: Never disable default outlines without replacing them with custom focus rings.
